# 📇 Business Card App - Jetpack Compose

A sleek and professional **Business Card Android Application** built with **Jetpack Compose**. This project demonstrates clean UI implementation, effective use of composable layouts, and custom styling in modern Android development.

## 🚀 Features
* **Modern UI Architecture:** Built entirely using Jetpack Compose declarative UI.
* **Layout Management:** Utilizing `Column`, `Row`, and `Arrangement` to create a perfectly balanced screen.
* **Theming & Design:** * Soft mint background for a modern look.
    * Custom icons for contact information (Phone, Email, Location).
    * Typography hierarchy to emphasize the name and role.
* **Vector Graphics:** Integration of Android Studio logo and Material icons.

## 🛠️ Technical Stack
* **Language:** Kotlin
* **Toolkit:** Jetpack Compose
* **Design Pattern:** Material Design 3
* **Minimum SDK:** 24+

## 📂 Project Structure
* **Profile Section:** Displays the logo, full name, and professional title.
* **Contact Section:** Bottom-aligned container with specific contact details and consistent iconography.
